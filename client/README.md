# Echo scenarios

A set of test scenarios for Echo server.

## Install

```
bundle install
```

## Usage

To run all scenarios, run:

    bin/rake scenario SERVER_URL=<ECHO_URL>

To run specific scenario, run:

    bin/rake TEST=scenario/<FILE> TESTOPTS="--verbose" SERVER_URL=<ECHO_URL>
